Introducing WINOPT - Window Optimization Tool

The tool enables the architects to come up with the most energy efficient design for the buildings during the early design phase. It uses GenOpt (optimization software) in the back-end in order to perform optimized number of simulations (can reduce millions of simulations to a few hundreds). The tool uses cluster for distributed computing and processing . Currently the tool is hosted at LBNL Lab, Berkley.
